# FUN-GUS: SILLIUM NETWORK TRADING CARDS
## Complete 12-Card Character Set

**Format:** Trading card set (standard 2.5" x 3.5" / 63mm x 88mm)
**Style:** Pixar-quality 3D character portrait on front; stats, facts, and lore on back
**Audience:** Ages 6–12 (collectible, educational, play-ready)
**Media Product:** List B — Trading Cards

---

## DESIGN SPECIFICATIONS

### Card Front Layout
- **Top Banner:** Character name in mycelium-thread font
- **Center:** Full character portrait (existing images from `/images/`)
- **Bottom-left:** Role icon (Guide, Scout, Messenger, etc.)
- **Bottom-right:** Kingdom badge (Fungal / Bacterial / Insect / Human)
- **Border:** Glowing network thread pattern — color matches character's signature color
- **Rarity indicator:** Corner glow (Common = green, Uncommon = blue, Rare = gold, Legendary = purple)

### Card Back Layout
- **Header:** Character name + species (real scientific name)
- **Stats Grid:** 5 stats rated 1–10 with visual bar
- **Special Ability:** Named power with one-sentence description
- **Real Science Fact:** One verified science fact in a callout box
- **Quote:** Character's signature line in italics
- **Series Logo:** Fun-Gus: The Forest Has WiFi (bottom)

### Color System
Each card's border and accent colors match the character's signature palette:
- Fun-Gus: Red-orange / amber
- Spora: Pink-white / soft gold
- Truff-Low: Deep purple / orange
- Lumos: Cyan-teal / blue-green
- Puff: Cream-white / golden
- Chanty: Apricot-gold
- Rhiza: White-to-brown gradient / gold
- Inky: White-to-purple-black gradient
- Stench: Olive-green / brown
- Birdie: Dark brown / amber
- Snapper: Blue-gray / silver
- Buzz: Gold / black stripes
