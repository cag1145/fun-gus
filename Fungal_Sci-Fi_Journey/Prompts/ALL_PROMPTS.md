## 1. FUN-GUS

**Role:** Main protagonist, network guide
**Species:** Amanita-style mushroom

**Style:** Pixar-quality 3D render, soft cinematic lighting, kid-friendly

**Character Description:**
A friendly anthropomorphic mushroom character with a large red-orange cap covered in white spots (amanita-style). The cap has glowing golden-orange gill patterns underneath that radiate warmth. Cream-colored round body with a gentle, welcoming face featuring large amber eyes with rosy cheeks. Warm smile showing approachability.

Wears a small blue sash/strap across body with a golden device attached (Sillium interface tool). Body has subtle glowing circuit-like patterns in warm gold/amber tones, suggesting connection to the network.

**Environment:** Enchanted forest floor with soft bokeh, other colorful mushrooms in background, mossy rocks, magical atmosphere.

**Lighting:** Warm bioluminescent glow from character, soft ambient forest light.

**Mood:** Joyful, inviting, trustworthy guide.

**Key Visual Rules:**
- Glow always directed downward toward soil (network connection)
- Never aggressive postures
- Warm white/gold internal glow suggests trust

## 4. SILLIUM (Environment)

**Role:** The mycelium network itself
**Type:** Environment/Setting (not a character)

**Style:** Cinematic 3D render, bioluminescent lighting, awe-inspiring scale

**Environment Description:**
A vast underground cavern system representing the Sillium network. Organic tunnel walls with flowing, curved surfaces like the inside of a living organism. Multiple mushrooms of varying sizes grow from walls and surfaces, each connected by glowing threads.

Dominant feature: Cyan/teal bioluminescent mycelium threads running throughout the space like fiber optic cables or neural pathways. These threads pulse with light, showing data transmission. Threads converge at mushroom nodes which act as routing stations.

The space has depth — tunnels branch off into darkness, suggesting the network extends infinitely. Some areas glow brighter (high traffic), others dim (dormant pathways).

**Lighting:** Primary cyan/teal glow from network threads. Secondary warm glow from mushroom caps (amber/gold gills). Dark shadows in recesses create mystery and scale.

**Mood:** Vast, alive, interconnected, technological yet organic.

**Key Visual Rules:**
- Cyan/teal = active communication
- Pulses move through threads = messages traveling
- Brighter nodes = high activity routing points
- This is a SYSTEM, not a character
- Scale should feel immense and awe-inspiring
