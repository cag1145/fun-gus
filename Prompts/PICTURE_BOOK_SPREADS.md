## PRODUCTION NOTES

### Generation Order
1. Character reference sheets (Birch, Fern, Ms. Hawthorn, Fun-Gus, Spora, Lumos, Puff, Chanty, Rhiza, Inky, Stench, Birdie, Snapper) — establish consistency
2. Cover — marketing asset, now features multiple characters
3. Hero spreads (Spread 2: Down to Sillium, Spread 13: All Together) — visual anchors
4. Underground character spreads as a batch (Spreads 3-12) — consistent cavern look
5. Above-ground spreads (Spread 1, Spread 14) — consistent forest lighting
6. Back matter infographic last

### Consistency Tips
- Generate all underground scenes in one session to maintain consistent cavern/network look
- Generate all above-ground scenes in one session for consistent forest lighting
- Keep the kids' clothing IDENTICAL in every image (yellow boots/purple jacket, green boots/brown vest)
- Each character should match their reference sheet exactly when they appear in scene prompts
- The fading thread (the seedling's connection) should be visually consistent across Spreads 3, 13 — same weak, sputtering quality until the climax

### Color Language (consistent across all spreads)
- **Cyan/teal** = network communication, active threads, SILLIUM
- **Amber/gold** = learning, growth, connection, Spora's glow, story-dust
- **Warm white** = trust, guidance, Fun-Gus's glow
- **Purple-black** = preservation, record-keeping (Inky only)
- **Cool blue-gray** = protection, strength (Snapper only)
- **Golden-amber** = storytelling, memory (Chanty)
- **Olive-green/brown** = Stench's unique palette

### Print Specifications
- Generate at highest resolution available (Midjourney: `--q 2`, DALL-E: 1024x1024 minimum)
- Final images need to be at least 3000x2400px for 10"x8" at 300dpi
- Leave space for text placement — generally bottom third or top quarter of each spread
- Full-bleed spreads need 0.125" extra on all edges (generate slightly larger, crop to trim)
